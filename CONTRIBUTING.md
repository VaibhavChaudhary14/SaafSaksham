# Contributing to SaafSaksham

Thanks for your interest in contributing to **SaafSaksham** 🎉  
This project exists to build real, measurable civic impact — and we’re excited to build it together.

---

## Ways to Contribute

You can help SaafSaksham in many ways:

- 🧑‍💻 Fix bugs or add features
- 🎨 Improve UI/UX and accessibility
- 🤖 Enhance AI/ML verification models
- 📊 Improve analytics and dashboards
- 📝 Improve documentation
- 💡 Propose new civic use cases

Clean, readable code > clever hacks

Security & privacy first

Civic impact must remain verifiable

Actual behavior
